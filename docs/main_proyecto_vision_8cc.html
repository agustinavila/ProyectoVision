Programa de ejemplo del uso de la clase AnalizadorSimetria.

Este es un programa de ejemplo mostrando algunas de las funcionalidades de la clase. Se puede cambiar durante ejecucion el tipo de feeder y de extractor de landmarks sin problemas. No se agregó la posibilidad de habilitar o deshabilitar el registro de video o landmarks, pero seria muy sencillo de implementar los metodos enableVideoLog, disableVideoLog, etc.

Parámetros
argc
argv
Devuelve
int

Hace referencia a DLIB, AnalizadorSimetria::getAsimetria(), AnalizadorSimetria::getLandmarks(), KINECTFEEDER, OPENCV, AnalizadorSimetria::setExtractor(), AnalizadorSimetria::setFeeder(), AnalizadorSimetria::step(), VIDEOFEEDER y WEBCAMFEEDER.
