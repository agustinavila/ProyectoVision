Detailed Description

Sobrecarga de la clase Feeder para proveer objetos Mat desde una kinect.

Por ahora, solo obtiene imágenes de la cámara RGB. Para definir el tipo de log a consola, se debe declarar una variable de entorno "LIBFREENECT2_LOGGER_LEVEL".
